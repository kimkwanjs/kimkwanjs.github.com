    개인프로젝트 로 클라이언트 와 서버 까지 의 전반적인 web 동작에 대한 이해를 돕기위해 제작. 클라이언트를 Vue.js 를 사용하여 SPA 제작. 평소좋아하던 게임이라는 장르를 JS를 이용해 만들 어 본 프로젝트.

    ㆍ 클라이언트 는 Vue.js 로 구현
    ㆍ 서버는 node.js - Express 로 구현
    ㆍ 유저간 통신은 Socket.io 를 사용
    ㆍ CSS 를 이용한 인터랙션 구현
